This package contains all the classes which handle directly the Umbra editor functions. This package contains three subpackages.
  • {@see umbra.editor.actions} which handles directly the GUI actions available in the bytecode editor
  • {@see umbra.editor.boogiepl} which handles the BoogiePL functionality
  • {@see umbra.editor.parsing} which handles the rudimentary parsing of the textual representation of the bytecode which is needed for colouring of the code
